# practice
Code Practice：Projects & Notes

这是一个 C 语言数据结构与算法练习仓库。

## 项目目标
- 系统练习经典数据结构与算法
- 提升 C 语言编程能力（指针、内存管理、模块化）
- 积累高质量代码项目经验

## 计划实现
- [ ] 单向链表 & 双向链表
- [ ] 栈 (Stack) 和 队列 (Queue)
- [ ] 二叉搜索树 (BST)
- [ ] 图的基本操作
- [ ] 哈希表
- [ ] 常见排序与搜索算法
- [ ] Big O 时间复杂度分析
